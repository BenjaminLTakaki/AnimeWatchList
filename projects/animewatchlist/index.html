AnimeWatchList Project

This is a Flask web application for tracking anime you've watched and want to watch. The app allows you to discover, search, and organize anime series.

Features of this application:

  • Discover popular anime from MyAnimeList
  • Search for specific anime titles
  • Mark anime as watched or not watched
  • Build and maintain your personal anime lists
  • View detailed information about each anime

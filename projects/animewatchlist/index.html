AnimeWatchList Project

This is a Flask web application for tracking anime you've watched and want to watch. The app allows you to discover, search, and organize anime series.

To run this project locally:

  1. Make sure you have Python and Flask installed
  2. Navigate to the project directory in your terminal
  3. Run pip install -r requirements.txt to install dependencies
  4. Start the app with python app.py
  5. Open your browser to http://localhost:5000

AnimeWatchList

A web application for tracking anime you've watched and want to watch. Built with Python Flask, this app allows users to discover, search, and categorize anime series.

Python Flask JavaScript API

AnimeWatchList allows users to:

  • Discover popular anime from MyAnimeList
  • Search for specific anime titles
  • Mark anime as watched or not watched
  • Maintain personal anime lists
